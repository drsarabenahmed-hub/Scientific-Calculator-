# Scientific-Calculator-
Scientific Calculator (Tkinter GUI)

This project is a Scientific Calculator built using Python and Tkinter.
It includes basic arithmetic operations as well as scientific functions such as sine, cosine, tangent, logarithms, square roots, and more.

Features:

User-friendly GUI interface using Tkinter

Basic operations: addition, subtraction, multiplication, division

Scientific functions:

Error handling for invalid inputs

Requirements :

Python 3.x

Tkinter (comes pre-installed with Python)

How to Run:

Download or clone the repository:

Open the folder containing the project.

Run the script:


Project Goal:

This project was created as a mini project to practice GUI programming in Python using Tkinter and to explore scientific function implementations.
